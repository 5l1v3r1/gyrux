# About Gyrux Language

    Gyrux Language is an expressive programming language with interactive CLI. 
    Gyrux has a lot of useful features such as facility to manipulate data structures, 
    object oriented programming, functional programming and others. 

***

# Getting started

## Linux

### amd64

> cd gyrux

> make linux-amd64

### arm64

> cd gyrux

> make linux-arm64

### i386

> cd gyrux

> make linux-i386

## macOS

### amd64

> cd gyrux

> make darwin-amd64

## Windows

### amd64

> cd gyrux

> make windows-amd64

### i386

> cd gyrux 

> make windows-i386

## FreeBSD

### amd64

> cd gyrux

> make freebsd-amd64

## OpenBSD 

### amd64

> cd gyrux

> make openbsd-amd64

## NetBSD

### amd64

> cd gyrux

> make netbsd-amd64
 
***
 
# Executing Gyrux Language

    To execute the Gyrux Language, you 
    should execute the following command.

> gyrux
